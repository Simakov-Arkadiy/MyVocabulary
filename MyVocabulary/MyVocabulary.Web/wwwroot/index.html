Мой словарный запас.

функции

  • записать новое слово
  • найти слово
  • задание для проверки

Приложение поможет запомнить выученные ранее тобой иностранные слова. Ты можешь выполнить запись нового слова с его переводом, можешь найти среди записаных слов нужное, также можешь получать задание для проверки знаний
